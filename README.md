## 🎯 Course Objectives

The course aims to enable students to:

- Understand the fundamental concepts of Artificial Intelligence.
- Model intelligent agents and problem-solving techniques.
- Apply uninformed and informed search algorithms.
- Represent knowledge using logical reasoning techniques.
- Introduce Machine Learning fundamentals.
- Develop awareness of ethical and responsible AI.

---

## 🎓 Course Outcomes

Upon successful completion of the course, students will be able to:

- Explain the principles and foundations of Artificial Intelligence.
- Apply AI search algorithms to solve computational problems.
- Represent knowledge using logical reasoning techniques.
- Explain Machine Learning fundamentals.
- Analyze AI applications and ethical considerations.
